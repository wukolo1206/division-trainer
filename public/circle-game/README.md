## index.html — 圓的秘密

**學習目標：** 透過探索，自己發現「所有半徑等長」和「直徑 = 半徑 × 2」

### 三段式流程

**① 畫圓（draw）**
- 從圓心往外拖曳
- 顯示動態圓規 V 形臂：轉軸、針尖（圓心）、筆尖
- 即時標示「開口 = 半徑 = X 公分」
- 樞紐位置跟著拖曳方向旋轉（垂直方向偏移）

**② 探索半徑（explore）**
- 點圓的邊緣，每次長出一條彩色半徑線 + 公分數標籤
- 點 4 次後出現發現卡：「量了 4 次，每次都一樣！」
- 下方彩色標籤列累積顯示所有量測結果

**③ 觀察直徑（diameter → done）**
- 按「繼續看直徑」→ 半徑動畫延伸，穿過圓心到對側
- 兩段各標「半徑 X 公分」，再顯示「直徑 Y 公分」
- 動畫結束後畫布狀態**持續保留**，直到使用者點「再畫一個圓」才清除

## draw.html — 圓規練習

**學習目標：** 模擬真實操作圓規的步驟，練習畫指定大小的圓

### 兩段式操作

**第一階段：設定開口（aim）**
- 畫布顯示目標虛線圓（粗藍色虛線，方便對準）
- 拖曳調整圓規開口大小
- 對準目標半徑（容許 ±0.45 公分）→ 自動 0.7 秒後確認
- 標示「開口 = 半徑 = X 公分」和「目前 X 公分（目標 Y）」

**第二階段：旋轉畫圓（trace）**
- 圓規 V 形臂跟著手指旋轉
- 筆尖沿固定半徑累積筆跡（offscreen canvas 疊加，不會閃爍）
- 底部進度條顯示完成百分比（0–100%）
- 覆蓋 320°+ 自動判定完成，補全缺口

**完成卡**
- 顯示半徑與直徑數值
- 按「下一題 →」才進入下一題（完成狀態不會自動消失）

## 技術備註

- 純 HTML + Canvas，無外部依賴，可直接在瀏覽器開啟
- 支援滑鼠與觸控（touchstart / touchmove / touchend）
- `draw.html` 使用 offscreen canvas 累積圓弧，避免每幀重畫造成筆跡消失
- 圓規樞紐位置公式：`pivot = CENTER + PIVOT_DIST × (拖曳方向垂直向量)`，確保 V 形開口永遠對著拖曳方向
- 兩頁互有連結（右上角導覽）
